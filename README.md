### [https://movies.listen-me.ru](https://movies.listen-me.ru)


#### Web сервис для поиска по базе фильмов.

#### Технологический стек:


HTML, CSS, БЭМ, JavaScript, React, express, mongoDB, webpack, npm, git, linux, bash
